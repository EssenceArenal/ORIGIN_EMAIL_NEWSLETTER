How to Send It

1

Choose your design

Open the HTML file that matches your organization's style. Preview it by double-clicking the file — it will open in your browser.

2

Customize it (optional)

Open the file in a text editor (Notepad on Windows, TextEdit on Mac). Find the sections marked with your content and update as needed. See "How to Customize" below.

3

Copy the HTML code

Select all the text in the file (Ctrl+A or Cmd+A) and copy it (Ctrl+C or Cmd+C).

4

Paste into your email platform

In Mailchimp, Klaviyo, or similar: create a new campaign, choose "Code your own" or "Custom HTML", and paste the code. Then send!

How to Customize

Open the HTML file in a text editor. Here are the most common things to change:

Change your organization name in the footer

Find this near the bottom of the file and replace the text:

Alliance network · <a href="#">Unsubscribe</a> → Change to your org name: Your Organization Name · <a href="YOUR_UNSUBSCRIBE_LINK">Unsubscribe</a>

Add a personal intro paragraph

Find the "What is Origin" section and add your own intro before the existing text:

<p class="body-p">Hi from [Your Org Name]! We're proud to be part of ORIGIN and wanted to share this with our community.</p>

Change the CTA button link

Search for href="https://origin-poster.onrender.com" and replace with your own registration link if needed.

Change colors (Design 3 and 5 only)

At the top of the CSS, find the :root block and change the color values:

:root { --gold: #c9a84c; /* ← change this to your accent color */ --black: #0a0a0a; /* ← change this to your brand dark color */ }

Change fonts

Replace the Google Fonts link at the top and update the font-family references in the CSS. Only do this if you're comfortable with HTML.

Need Help?

If you get stuck, contact the ORIGIN team. For simple text changes (your org name, a paragraph), you don't need any technical skills — just find the text and replace it. For color or font changes, you may want to ask someone comfortable with HTML.
